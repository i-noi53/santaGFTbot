# 🎮 Telegram Key Word Game Bot

## 📌 О проекте

Это Telegram-бот-игра, в которой пользователи должны угадывать спрятанные слова.

Администратор может задавать 3 ключевых слова, а игроки должны найти их, отправляя сообщения в чат.

### ✨ Возможности

- ✅ Запуск и остановка игры
- ✅ Добавление новых игровых слов (`/add_game`)
- ✅ Подсказка при появлении ключевого слова в сообщении
- ✅ Засчитывание слов внутри предложений
- ✅ Победа только при вводе всех 3 слов подряд в правильном порядке
- ✅ Индивидуальный прогресс игроков
- ✅ Безопасное хранение токена через `.env`

Установка:

- pip install aiogram python-dotenv
- Скопировать bot.py
